## THE TILT TEST

This product should feel closer to:

| Closer To | Not This |
|-----------|----------|
| "Something is happening" | "Everything is okay" |
| "Act now" | "Take your time" |
| "Money is moving" | "Your finances are secure" |
| "People are nearby" | "Welcome to our community" |
| "You're late" | "You're early" |

If a design feels:
- Calm
- Polished but passive
- Neutral
- Balanced
- Reassuring

**It has failed the axis test, even if it passes UAP-6.**

---

## AXIS APPLICATION BY ELEMENT

### Background
```
TILT: Energy, not emptiness

NOT: Static gradient that says "premium fintech"
YES: Subtle motion that says "system is running"

The background should imply:
"This place was active before you arrived"
```

### Brand Mark
```
TILT: Consequence, not focus

NOT: Hero logo in center spotlight
YES: Logo that emerges from the environment

The brand should feel like:
"This place has a name" — not "look at our logo"
```

### Headline
```
TILT: Declaration, not description

NOT: "Get things done. Get paid." (describes function)
YES: "Turn time into money." (declares outcome)
YES: "There's work nearby." (creates urgency)
YES: "You're minutes from your first task." (implies action)

The headline should feel like:
"This is happening" — not "this is what we do"
```

### CTA
```
TILT: Entry, not start

NOT: "Get Started" (generic, passive)
YES: "Find work near me" (action + location)
YES: "Post a task now" (action + immediacy)
YES: "Start earning today" (outcome + time)

The CTA should feel like:
"Enter the system" — not "begin a process"
```

### Motion
```
TILT: Directional, not decorative

NOT: Ambient glow that loops forever
YES: Energy that flows toward the CTA
YES: Pulse that implies heartbeat
YES: Particles that suggest rising opportunity

Motion should imply:
"Something is pulling you forward"
```

## TRADEOFF DECISION MATRIX

When a design choice requires tradeoff:

| If Choice Is Between | Choose |
|---------------------|--------|
| Calm vs. Energetic | Energetic |
| Spacious vs. Dense | Dense (with hierarchy) |
| Reassuring vs. Urgent | Urgent |
| Safe vs. Bold | Bold |
| Polished vs. Raw | Polished but tilted |
| Balanced vs. Directional | Directional |
| Generic vs. Distinctive | Distinctive |
